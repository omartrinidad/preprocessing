Welcome

This is the website for the Collection of Preprocessed Digital Mammograms, a collection of mammographies publicly available. Each mammogram was preprocessed with different algorithms. If you want to know more details about the algorithms you can access to the article.

If you made use of this work please cite the next work: "A New Collection of Preprocessed Digital Mammograms".
